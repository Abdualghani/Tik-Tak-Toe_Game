# Tic-Tac-Toe Game

A clean, responsive, browser-based Tic-Tac-Toe game.

## Description
This is a lightweight, interactive web application that allows two players to play the classic game of Tic-Tac-Toe directly in their web browser. No installation or heavy setup required!

## Features
* **Two-Player Gameplay:** Classic turn-based system (`X` vs `O`).
* **Win/Tie Detection:** Automatically identifies when a player wins or when the game ends in a draw.
* **Responsive Design:** Works smoothly across both desktop screens and mobile devices.
* **Instant Reset:** Clear the board instantly to start a new round.

## Technologies Used
* **HTML5:** For the structural layout of the game grid.
* **CSS3:** For modern styling, animations, and responsive layout.
* **JavaScript:** Handles the core game logic, turn tracking, and win conditions.
